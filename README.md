# MKI 福利厚生ナビゲーター

三井情報（MKI）の社員向け福利厚生を簡単に検索・確認できるWebアプリケーションです。

**※ 本アプリケーションは、三井情報株式会社の公式情報に基づいて作成された福利厚生検索ツールです。**

## 🎯 目的

社員が福利厚生を十分に把握していないという課題を解決するため、以下の機能を提供します：
- 社員情報に基づいた利用可能な福利厚生の自動表示
- 会話ベースでの福利厚生検索

## 🚀 使い方

### 起動方法

1. `index.html` をWebブラウザで開いてください
2. インターネット接続は不要です（スタンドアロンで動作します）

### 機能

#### 1. 基本情報から探す

社員情報を入力することで、利用可能な福利厚生が自動的に表示されます。

**入力項目：**
- お名前
- 住所
- 年齢
- 勤続年数
- 結婚状況（独身/既婚）
- お子様の有無と人数

**例：**
- 勤続5年以上の社員には「リフレッシュ休暇」が表示されます
- 既婚者には「結婚祝い金」が表示されます
- お子様がいる方には「育児支援手当」「出産祝い金」などが表示されます

#### 2. 会話で探す

チャット機能を使って、自然な言葉で福利厚生を検索できます。

**使用例：**
- 「結婚しました」→ 結婚祝い金、住宅補助、綱町三井倶楽部などを表示
- 「子供が生まれました」→ 出産祝い金、育児休業制度などを表示
- 「引っ越しします」→ 住宅補助などを表示
- 「健康診断を受けたい」→ 健康診断、こころとからだの健康相談などを表示
- 「保険について知りたい」→ 各種保険制度を表示
- 「旅行に行きたい」→ ITS旅行パック、リフレッシュ休暇などを表示
- 「介護が必要」→ 介護休業制度、健康相談などを表示

#### 3. 検索分析（NEW!）

チャット機能での検索履歴を自動的に集計・分析し、社員が求めている福利厚生を可視化します。

**表示内容：**
- **統計情報**：総検索回数、検索された福利厚生の種類、最終検索日時
- **人気ランキング TOP10**：最も検索された福利厚生をランキング形式で表示
- **検索キーワード履歴**：最近の検索キーワードを時系列で表示
- **カテゴリ別検索状況**：どのカテゴリの福利厚生が人気かをバーチャートで可視化

**人事部向け機能：**
- 📥 **データエクスポート**：分析データをJSONファイルとしてダウンロード
- 🗑️ **データクリア**：検索履歴をリセット

**活用例：**
- 社員がどの福利厚生に興味を持っているか把握
- 認知度の低い福利厚生を特定
- 新しい福利厚生の企画に活用
- 福利厚生の周知活動の効果測定

## 📋 登録されている福利厚生（24種類）- MKI実際の制度

### 給与・手当系
- 💰 賞与（年2回：夏・冬）
- 🚃 通勤費・交通費支給（全額支給）
- 💻 在宅勤務手当
- 🏠 住宅補助
- 🏢 常駐手当
- ✈️ リフレッシュ休暇手当（勤続10年経過後、5年ごとに5日間＋手当）
- 🤝 採用貢献手当（リファラル採用）

### 将来設計系
- 💼 ライフデザイン手当（退職金制度）- 確定拠出年金＋前払退職金
- 💵 財形貯蓄（一般・年金・住宅）

### ライフイベント系
- 💍 慶弔見舞金（結婚祝い金）
- 👶 慶弔見舞金（出産祝い金）
- 🕊️ 慶弔見舞金（弔慰金・見舞金）
- 👨‍👩‍👧 介護休業制度（介護短時間勤務・介護休暇）

### 育児系
- 👶 育児休業制度（育児短時間勤務・子の看護休暇）
- 🎓 遺児育英資金

### 健康系
- 🏥 健康診断（全額会社負担、人間ドックも選択可）
- 💚 こころとからだの健康相談（無料カウンセリング）

### 保険系
- 🛡️ 総合福祉団体定期保険（会社全額負担）
- 📋 団体扱い生命保険・損害保険（団体割引）
- 🛡️ 総合保障保険・スーパー所得補償保険（三井物産グループオリジナル）
- 🏥 健康保険・厚生年金保険（各種社会保険完備）

### レジャー・施設系
- 🏖️ 関東ITソフトウェア健康保険組合（ITS）- 保養施設、スポーツ施設、ゴルフ場、オートキャンプ場、ラフォーレ倶楽部、ITS旅行パックなど
- 🏛️ 綱町三井倶楽部（会員制倶楽部、宴会・パーティー・ウェディング）
- 🍽️ 新宿三井クラブ（新宿三井ビル54階の会員制レストラン）

## 🔧 カスタマイズ方法

### 福利厚生の追加・編集

`script.js` の `benefitsDatabase` 配列に新しい福利厚生を追加できます。

```javascript
{
    id: 16,
    icon: "🎁",
    name: "新しい福利厚生",
    description: "説明文",
    amount: "金額",
    conditions: {
        minYears: 0,        // 最低勤続年数
        marital: null,      // "married" or "single" or null
        children: null      // "yes" or "no" or null
    },
    keywords: ["キーワード1", "キーワード2"],
    category: "カテゴリ"
}
```

1. **最新情報の確認**: 福利厚生の内容や金額は変更される可能性があります。最新の情報は人事部または社内ポータルサイトでご確認ください。
2. **申請方法**: 実際の申請方法や必要書類については、人事部にお問い合わせください。
3. **個人情報**: 本アプリケーションで入力した情報は保存されません（ブラウザ上でのみ動作）。
4. **公式情報**: 本アプリケーションは2025年時点の公開情報に基づいています。
